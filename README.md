# Vortex
A powerful markdown presentation tool.

# Features
* Markdown Syntax
* Beautiful Animates & Themes
* PDF Export
* Syntax Highlight
* Flowchart & Sequence Diagram & Gant Diagram
* Customize By CSS
* Support Linux, OSX, Windows

# Contribute
## Install dependencies
```
npm install
```
## Start
```
npm start
```
## Package
You can change the release platform in config.js.
```
npm run release
```
